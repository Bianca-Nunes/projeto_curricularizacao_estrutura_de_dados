# Controle de Insumos e Precificação

Sistema desenvolvido para auxiliar pequenos empreendedores no controle de insumos, gerenciamento de estoque, cálculo de custos de produção e formação de preços de venda. O projeto aplica conceitos de Estrutura de Dados, utilizando listas encadeadas, pilha, fila, tabela hash, algoritmos de busca e ordenação para otimizar a gestão de produtos, receitas e reposição de estoque.
